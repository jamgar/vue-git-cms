# vue-git-cms

This is a website and cms built with Vue. It inspired by [NetlifyCMS](https://www.netlifycms.org/).

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
